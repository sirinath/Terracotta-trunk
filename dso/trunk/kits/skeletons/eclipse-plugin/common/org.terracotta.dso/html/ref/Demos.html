Terracotta DSO Demonstration Applications

Terracotta provides a set of prebuilt applications meant to demonstrate the important features of DSO.

JTable

The JTable application is the DSO analog of the infamous helloworld sample.

Shared Editor

The SharedEditor application models a simple shared drawing area demonstrating one possible use-case for DSO.

Loading JTable and SharedEditor Demos into Eclipse


1. Under the file menu select import...




2. Browse to select the jtable directory under demo.




3. Right-Click the newly imported project to display the context menu. Select Terracotta/Add Terracotta Nature. This essentially defines an Eclipse project type.




4. Load the pre-configured domain config. tc-config.xml which should show up by default.




5. Again load the Terracotta context menu to start and stop the server. The server must be running before the demos load. If the programs are running correctly, you will see status messages in the Eclipse console
