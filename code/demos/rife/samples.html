Terracotta RIFE

Examples of using Terracotta to cluster the JVM

Sample Applications

  • RIFE Continuations
    The RIFE example shows how native Java web continuations can be clustered with Terracotta DSO. This three-step check-out process uses continuations for flow control, the related server-side state is automatically shared across the nodes in the cluster.
